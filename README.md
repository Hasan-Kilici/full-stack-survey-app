VSC'de çalıştırmak için 2 tane terminal açın
birine
```sh
cd back-end
go get
go run .
```

diğerine
```sh
cd front-end
npm i
npm i js-cookie
npm i svelte-routing
npm run dev
```
yazın

PostgreSQL bilgisayarınızda kuruluysa SQL dosyasından gerekli query'i alın
